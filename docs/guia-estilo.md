# 🎨 Guia de Estilos (Design System)

Este documento contém a referência visual e de usabilidade para as interfaces do projeto Desenrola, extraídos do Figma da equipe.

## ✨ Paleta de Cores

As cores abaixo definem a identidade visual da plataforma, variando entre tons vibrantes para chamadas de ação e escalas de cinza para fundos e textos.

- **Destaque Principal (Amarelo):** `#EDBD32` (100%)
- **Destaque Suave 1:** `#EDBD32` (30%)
- **Destaque Suave 2:** `#EDBD32` (20%)

- **Preto (Texto Primário / Alto Contraste):** `#000000`
- **Cinza Escuro (Textos Secundários):** `#4A5565`
- **Cinza Médio (Bordas / Ícones):** `#6A7282`
- **Cinza Claro 1 (Fundos Secundários):** `#E6E6E6`
- **Cinza Claro 2 (Fundos de Seção):** `#F5F5F5`
- **Branco (Superfícies / Cards):** `#FFFFFF`

## 🔤 Tipografia

A família tipográfica utilizada na interface é a **Inter (Sans Serif)**.
- **Tamanho Base (Base Value):** `16px`
- **Escala Tipográfica (Scale):** `1.25`

### Escala de Tamanhos:
- **61px** / `3.813rem` (Títulos Extra Grandes / Hero)
- **49px** / `3.063rem` (Títulos H1)
- **39px** / `2.438rem` (Títulos H2)
- **31px** / `1.938rem` (Títulos H3)
- **25px** / `1.563rem` (Títulos H4 / Subtítulos)
- **20px** / `1.250rem` (Títulos H5 / Destaques)
- **16px** / `1.000rem` (Corpo de Texto Padrão / Parágrafos)
- **13px** / `0.813rem` (Textos Auxiliares / Legendas)
- **10px** / `0.625rem` (Textos Mínimos / Tags)

---
> *Nota: Os valores de `rem` são calculados assumindo o tamanho base de fonte do navegador como 16px.*
